# My keyboard layout for the [Bastyl-Mini](https://bastardkb.com/keebs/)

## BASE: Qwerty with [home row mods](https://precondition.github.io/home-row-mods)

### RGB

- On startup, if enabled, the board will cycle through the entire hue wheel, starting and ending on the default layer color

### Modifiers

- `EMAIL`: print email address
- `LAUNCH`: program launcher
- `BROWSR`: browser
- `LOCK`: lock screen

### Tap dances

- `OSM_CAPS`: one shot mod caps lock (the next word is all caps)
- `DOT` / `QUES` / `EXLM`: next sentence tap dance i.e. (`.` / `?` / `!`), space and capitalize next letter.
- `Shift + L(1,Bsp) = Delete`

```
,-----------------------------------------------------.        ,-----------------------------------------------------.
| EMAIL  |   q    |   w    |   e    |   r    |   t    |        |   y    |   u    |   i    |   o    |   p    |  L(5)  |
|--------+--------+--------+--------+--------+--------|        |--------+--------+--------+--------+--------+--------|
| LAUNCH | Alt(a) | Ctl(s) | Gui(d) |Shift(f)|   g    |        |   h    |Shift(j)| Gui(k) | Ctl(l) |  ;  :  |  ' "   |
|--------+--------+--------+--------+--------+--------|        |--------+--------+--------+--------+--------+--------|
| BROWSR |   z    |   x    |   c    |   v    |   b    |        |   n    |   m    |  ,  <  | DOT  > | / QUES |  LOCK  |
'--------------------------+--------+--------+--------'        '--------+--------+--------+--------------------------'
                           |L(4,Tab)|                                            |L(3,Esc)|
                           '--------+--------+                          +--------+--------'
                                    |L(1,Bsp)|                          |L(2,Spc)|
                                    '--------+--------+        '--------+--------'
                                             |OSM_CAPS|        | Enter  |
                                             +--------+        '--------'
```

- `FILE MAN`: file manager (lf)
- `PRT_CLIP`: print clipboard contents with notify-send
